# openwrt-docs4ai Release Tree

This release tree packages OpenWrt programming references for AI tools and human operators.

Start with [llms.txt](./llms.txt) for root routing, [AGENTS.md](./AGENTS.md) for agent-oriented guidance, or a module [map.md](./ucode/map.md) for structured navigation.

Each module exposes the same authoritative programming material in two forms:

- `bundled-reference.md` for broad context
- `chunked-reference/` for targeted lookups
